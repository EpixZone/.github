## What is Epix?

Epix is building the infrastructure for a decentralized world, built on two pillars:

- **⛓️ [EpixChain](https://github.com/EpixZone/EpixChain)** — a high-performance, EVM-compatible Layer 1 built on the Cosmos SDK. Full Ethereum tooling compatibility, IBC cross-chain transfers, native staking and on-chain governance, and a dynamic emission model capped at 42 billion EPIX.
- **🌐 [EpixNet](https://github.com/EpixZone/EpixNet)** — a peer-to-peer web where sites (*xites*) live on people's own computers instead of central servers. Every visitor helps host what they browse, with Tor and I2P privacy built in and on by default.

Together they form a full stack for decentralized applications: a chain for value and governance, and a network for content and communication, with the **xID** identity system linking the two.
