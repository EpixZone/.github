# .github

Organization-wide defaults for [EpixZone](https://github.com/EpixZone).

This repository provides:

- [`profile/README.md`](profile/README.md) — the organization profile shown at [github.com/EpixZone](https://github.com/EpixZone)
- [`CONTRIBUTING.md`](CONTRIBUTING.md) — default contributing guidelines
- [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) — community code of conduct
- [`SECURITY.md`](SECURITY.md) — security policy and private vulnerability reporting
- [`SUPPORT.md`](SUPPORT.md) — where to get help
- [`ISSUE_TEMPLATE/`](ISSUE_TEMPLATE/) — default issue templates
- [`PULL_REQUEST_TEMPLATE.md`](PULL_REQUEST_TEMPLATE.md) — default pull request template

These files apply to every EpixZone repository that doesn't define its own.
