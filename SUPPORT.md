# Getting Help

Thanks for using Epix! Here's where to go depending on what you need.

## 💬 Community chat

The fastest way to get help is the community [Discord](https://discord.gg/bF2GKHgrfv) — questions about running a node, building a xite, wallet setup, staking, and everything else.

## 🐛 Bug reports & feature requests

Open an issue on the relevant repository:

- Chain, tokenomics, RPC, precompiles → [EpixChain](https://github.com/EpixZone/EpixChain/issues)
- Node, browser, network, dashboard → [EpixNet](https://github.com/EpixZone/EpixNet/issues)
- Wallet → [epix-wallet](https://github.com/EpixZone/epix-wallet/issues)
- A specific app (Talk, Mail, Post, Blog, Wiki, Sites) → that xite's repository

Please use the issue templates and include versions, your platform, and reproduction steps.

## 📖 Documentation

- EpixChain website: [epix.zone](https://epix.zone)
- EpixNet website: [epixnet.io](https://epixnet.io)
- EpixNet docs: [EpixNetDocs](https://github.com/EpixZone/EpixNetDocs) and the install guides in the [EpixNet repo](https://github.com/EpixZone/EpixNet/tree/main/docs/install)
- Chain quick-start: the [EpixChain README](https://github.com/EpixZone/EpixChain#readme)

## 🔒 Security issues

**Never file security vulnerabilities as public issues.** See our [security policy](SECURITY.md) for private reporting channels.
